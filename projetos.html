Projetos

Money Mate - Sistema gerenciador de finanças pessoais

O projeto propõe um sistema web que permite a gestão intuitiva e eficiente de recursos orçamentários e metas financeiras em vista da dificuldade encontrada nesse processo, devido à falta de ferramentas adequadas. Entre os recursos desta aplicação estão: gestão de despesas, receitas, transações, metas e categorias financeiras, além da exibição de relatórios financeiros mensais. O objetivo é simplificar a organização da situação monetária de cada pessoa ou grupo familiar. O desenvolvimento foi iniciado com o levantamento análise e documentação de requisitos, além da elaboração de diagramas, de caso de uso, de classes e de atividades. A codificação envolveu o uso de linguagens como HTML, CSS e JavaScript, além do framework Bootstrap no frontend e o framework Django no backend. O armazenamento de dados foi realizado com o PostgreSQL. É importante ressaltar que este projeto ainda está em fase de protótipo e, portanto, ainda não foi validado com usuários reais.

CliniTech - Sistema de gestão de clínicas médicas

Para auxiliar as clínicas médicas e seus funcionários, o projeto propõe a construção de um site com funcionalidades como gerenciamento de consultas médicas, prontuários e pacientes, além do controle de estoque de insumos. O sistema visa tornar o gerenciamento mais intuitivo, eficiente e organizado, centralizando informações e facilitando o acesso e a comunicação entre pacientes, médicos e secretários.
